Job01

Pourquoi les balises "title" et "meta" sont-elles essentielles pour le SEO ?!

En HTML, le doctype est le préambule "!DOCTYPE html" requis en haut de tous les documents. Son seul but est d'empêcher un navigateur de passer en soi-disant "quirks mode" lors du rendu d'un document ; c'est-à-dire que le doctype "!DOCTYPE html" garantit que le navigateur fait de son mieux pour suivre les spécifications pertinentes, plutôt que d'utiliser un mode de rendu différent incompatible avec certaines spécifications. (Source Mozilla Developer)

Quel impact ont-elles sur la perception d'un moteur de recherche vis-à-vis de votre site ?

Les balises titre sont les balises principales. Vous devez leur apporter beaucoup d’attention. Elles ont un réel impact sur votre SEO et le positionnement de vos pages. De plus, elles sont les seules balises à être visibles par vos visiteurs, en haut de votre navigateur. Il est donc nécessaire de les remplir de la bonne façon. Elles ont le bénéfice de décrire précisément le sujet de votre page.
La meta description est une donnée importante à insérer dans votre HTML. Elle décrit aux moteurs de recherche en une courte description le sujet de votre page. Cette méta va plus loin que la balise titre en proposant d’avantage de contenu à vos visiteurs dans les SERP. Même si Google a démontré que les mots-clés renseignés dans cette description ne sont pas facteurs de positionnement, ils peuvent inciter vos visiteurs à cliquer sur votre site à travers les SERP si ces mots-clés sont pertinents. La meta description n’est donc pas importante pour ces mots-clés mais pour son sens dans son entièreté.(Source: Oncrawl)

Quel est le rôle de la déclaration DOCTYPE en HTML ?

La déclaration "!DOCTYPE" définit la version de HTML, qui est utilisée dans ce document. L'annonce de la version est une condition préalable, car le navigateur doit comprendre comment interpréter la page. Si la déclaration n'est pas disponible, le navigateur peut ne pas afficher le contenu de la page Web correctement. La déclaration "!DOCTYPE" est indiqué avant le tag "html". La déclaration n'est pas sensible à la casse. (Source W3docs)

Job02

Pourquoi est-il important d'utiliser des balises sémantiques en HTML ?

Les balises sémantiques permettent de renseigner les robots des moteurs de recherche quant à l’importance et la nature des éléments d’une page. Ce sont des éléments de code qui permettent de structurer une page pour la rendre claire et lisible aux yeux de Google

En quoi cela améliore-t-il le référencement et l'accessibilité ?

Ces balises sémantiques viennent structurer et hiérarchiser le contenu d’une page. Ainsi, elles permettent aux robots des moteurs de recherche de comprendre rapidement le sujet dont traite cette page, ainsi que sa structure. L’indexation des pages dont le code est structuré autour de ces balises est donc favorisée, et leur positionnement amélioré. Cela explique pourquoi le balisage sémantique occupe une place non négligeable dans la stratégie SEO d’un site web.

Comment les attributs name des champs de formulaire sont-ils utilisés lors de la soumission d'un formulaire ?

Les attributs "name" dans les champs de formulaire HTML servent à identifier les données saisies par l'utilisateur. Lorsque le formulaire est soumis, ces noms sont utilisés pour transmettre les données au serveur. Le serveur peut ensuite les récupérer en utilisant ces noms pour un traitement ultérieur, comme la validation ou l'enregistrement dans une base de données. En résumé, les attributs "name" sont des étiquettes qui lient les données aux champs et permettent leur traitement côté serveur lors de la soumission du formulaire.

Pourquoi est-il crucial d'ajouter un attribut alt à vos images ?

La fonction première de l’attribut alt est de permettre aux personnes malvoyantes de savoir à quoi correspond votre image. C’est, en effet, cette expression qui va être lue par les logiciels de lecture d’écran. Cet attribut présente également une utilité pour l’ensemble des internautes. C’est encore cette expression qui apparaîtra si le navigateur web de l’internaute n’arrive pas à charger votre image ou si votre site est trop lent.

(Source: Viaprestige-agency.com)

Comment cela influence-t-il l'accessibilité et le SEO de votre site ?

Utiliser l’attribut alt est aussi une façon de positionner son image dans les résultats de Google Images. Or il ne faut pas sous-estimer le trafic généré par cet outil. Les internautes ne se contentent plus de la recherche basique et n’hésitent pas à chercher du contenu au sein de Google Images. (Source: Viaprestige-agency.com)

Quelle est l'importance des liens internes pour le SEO ?

Le lien interne relie une page d’un site avec une autre page du même site ; il favorise la navigation des internautes sur le site mais aussi des robots de Google ! (Source: anthedesign.fr)

Comment influencent-ils le parcours de l'utilisateur ?

La création de liens internes offre la possibilité aux internautes de passer d’une page à une autre page traitant de sujets complémentaires. Ceux-ci peuvent alors découvrir votre site en profondeur et accéder à des informations disponibles sur d’autres pages du site. Les visiteurs apprécient particulièrement cela puisqu’il leur facilite la recherche et leur fait gagner du temps. (Source: anthedesign.fr)

Comment différenciez-vous un lien interne d'un lien externe ?

Un lien interne c’est quand vous faites un lien sur votre site vers une page interne de votre site. Donc c’est un lien sur votre site vers votre site. Un lien externe, c’est quand vous avez un lien depuis un site extérieur au vote qui fait un lien vers votre site. (Source: referenseo.com)

Liste ordonnée

  1. Mettre 200g de farine dans le saladier
  2. Casser 3 oeufs et les verser dans le saladier
  3. Rajouter 250ml de lait et mélanger le contenu du saladier avec un batteur
  4. Verser 50g de sucre puis mélanger
  5. Laisser reposer une heure
  6. Préchauffer le four à 250°C
  7. Verser le contenu du saladier dans plat et laisser au four pendant 50 minutes
  8. Laisser refroidir et décorer selon vos envies (Chantilly, vermicelles pour gâteau...)

La liste ordonnée permet de spécifier un classement par étapes, la numérotation est alors automatique.

Quelle est la différence entre les balises strong et b en termes de sémantique ?

La balise strong ou "fort" en français est une balise HTML sémantique, c'est à dire qu'elle a pour fonction de signifier que le contenu qui s'y trouve est de "haute importance" pour le sens du texte. Par défaut cette balise met en caractère gras les lettres qui y sont contenus. La balise b pour bold est une balise de style, qui met en gras les mots ou lettre qu'elle encadre. (Source: primaweb.fr)

Où est-il préférable de placer les balises "script" dans une page HTML pour une performance optimale ? et pourquoi c’est optimal ?

Emplacement des balises "script"

Techniquement, les balises "script" et "/script" peuvent être placées n'importe où dans la page Web. Cependant, il est préférable de choisir un emplacement qui tient compte de deux aspects importants : Le code JavaScript doit être exécuté seulement lorsque toutes les balises HTML seront chargées en mémoire. On dira que le DOM (Document Object Model) est entièrement chargé. Ceci asssure que le code JavaScript pourra effectuer les manipulations du DOM requises, par exemple ajuster une balise par programmation ou encore affecter du code à un événement rattaché à une balise. Le chargement du JavaScript ne doit pas ralentir l'affichage du code HTML. Le rendu de la page Web doit être aussi rapide que possible. C'est pourquoi les balises "script"" doivent être placées à l'un de ces endroits : dans la section "head" avec l'attribut defer (seulement si le code est dans un fichier .js) juste avant le "/body" avant le "/body", avec association du code JavaScript à l'évènement DOMContentLoaded
